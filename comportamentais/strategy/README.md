## A visão geral do padrão
Defini uma família de algoritmos que são colocadas em classes diferentes e permite que o cliente possa usar qualquer uma delas podendo também alterar. Em outras palavras, é um padrão comportamental que vai focar na execução de uma mesma coisa em classes diferentes 

## Como o padrão faz isso?
Extrai os códigos que fazem coisas específicas de maneiras diferentes e os coloca em classes denominadas estratégias.
